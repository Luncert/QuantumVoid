# Quantum Void

Quantum Void is a theme extension for the [Zed](https://zed.dev) code editor, offering both dark and light color schemes designed for a visually appealing and comfortable coding experience.

## Features

- Two themes: **Quantum Void Dark** and **Quantum Void Light**
- Carefully selected color palette for syntax highlighting and UI elements
- Designed for readability and reduced eye strain
